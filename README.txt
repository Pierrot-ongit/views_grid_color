------------
Description
------------

Un module offrant une interface utilisateur pour changer la couleur d'arrière plan (background-color en CSS) d'une grille générée par le module views.
Pour l'instant le module est assez limité. Il visait à répondre à un besoin client spécifique, afin de régulièrement changer les couleurs de la grille de produit d'un site d'e-commerce (lors de promotions).
Cela ne fonctionne qu'avec des Views au format Grille(Grid).
Cela ne fonctionne qu'avec une seule View à la fois.
Vous ne pouvez changez que les couleurs d'arrière plan.

Mais le module peut servir d'ébauche ou d'exemple si jamais vous avez besoin d'offrir une interface utilisateur pour changer les apparences d'une View.



------------
Requirements
------------

- Drupal 7



------------
Usage
------------

Installer et activer le module comme un module Drupal 7 ordinaire.

La page de configuration se situe à :
FR: administration > Configuration > Interface Utilisateur > Grille Couleurs Configuration

URL : votresite/admin/config/user-interface/views_grid_color

Elle requiert les droits d'administer la configuration du site pour y accéder.
